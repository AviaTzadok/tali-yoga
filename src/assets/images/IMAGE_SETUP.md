## Image Recommendations:
- **Format:** JPG or PNG
- **Size:** 
  - Hero image: 1920x1080px or larger (will be cropped/centered)
  - Gallery images: 800x600px or larger (will be automatically resized)
- **Optimization:** Compress images for web to reduce file size while maintaining quality

## After Adding Images:
1. The hero background will automatically use the studio interior image
2. All gallery images will display in the gallery section
3. Images will be optimized by Angular during build
